About This Website

This site started as a playground for learning the absolute basics of the web stack and has grown into a living archive of experiments, prototypes, and notes from the journey.

Development Story

The project kicked off with static HTML pages that documented each small breakthrough. As new ideas surfaced, the site absorbed them: stylesheets were refactored, JavaScript utilities were shared across pages, and an embedded React sandbox was introduced to document modern component explorations. Every branch of the repository doubles as documentation of what was tried, why it mattered, and what was left behind for future reflection.

Documentation & Resources

  • Project source, issues, and dev logs live in the GitHub repository.
  • Historical context for the "First JS" prototype, including notes on the original vanilla JavaScript widgets, resides in the documents section.
  • Experiments with authentication, security headers, and testing are captured across the tests/ folder and the linked reports.

What You Can Explore
